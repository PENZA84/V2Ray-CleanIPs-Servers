## 📌 How to use

### 🇮🇷 فارسی
1. یکی از لینک‌های بالا را کپی کنید.
2. در اپلیکیشن مورد نظر (v2rayNG / Hiddify / FoXray / NekoBox)
3. گزینه **Import from URL** یا **Add Subscription** را بزنید.
4. لینک را Paste کنید و آپدیت بگیرید ✅

---

### 🇺🇸 English
1. Copy one of the subscription links above.
2. Open your client app (v2rayNG / Hiddify / FoXray / NekoBox).
3. Choose **Import from URL** or **Add Subscription**.
4. Paste the link and update.
